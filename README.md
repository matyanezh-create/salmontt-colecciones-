# salmontt-colecciones-
Aplicación Java que carga información de centros de cultivo desde un archivo centros.txt, crea objetos de tipo CentroCultivo y los almacena en un ArrayList.
Luego muestra todos los centros cargados y aplica un filtro para identificar aquellos con mayor producción.

Estructura del proyecto:

⦁	model: Clase CentroCultivo.

⦁	data: Clase GestorDatos (lee el archivo y filtra datos).

⦁	ui: Clase Main (ejecuta el programa).

⦁	resources: Archivo centros.txt.

Para ejecutar: abrir el proyecto en IntelliJ y correr ui.Main.
